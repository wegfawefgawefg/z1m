# Dungeon Catalog

This is the campaign content checklist for the nine dungeons.

The point is not just “room data loads.” Each dungeon needs:

- correct entrance/exit flow
- correct boss and item
- staircase / transport / shutter / puzzle scripts
- enough room feature fidelity that progression is completable

- push-block secrets
- bombable walls
- locked doors
- shutter doors
- transport stairs
- item cellars / item rooms
- map / compass rooms
- old man / hint rooms
- enemy clear triggers
- kill-all-for-key/item triggers

All of the above still need concrete runtime representations.
